## APK打包发布
  * 方法一 ： http://flutter.liulongbin.top/
  * 方法二 ： 
    * 创建mykey.jks 签名文件
    * 配置 android/app/build.gradle 
    * 运行 flutter build apk
    * 打包好的发布APK位于 <app dir>/build/app/outputs/apk/app-release.apk
